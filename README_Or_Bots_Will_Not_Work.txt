1) Install TamperMonkey from https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=en
2) Install NeyBotsScript_Olaf4Snow_v2.js script into TamperMonkey
3) Go to CellCraft.io or Play.Agario0.com
4) Click Start-NeyBots-Olaf4Snow-v2.bat to open command window
5) Join server and click "Start" button and wait for bots to connect

Bots not connecting?
1) Click Update-Proxy.bat to automatically refresh proxy.txt with new Socks 5 proxies
2) Restart bots (close command window and then click Start-NeyBots-Olaf4Snow-v2.bat again)
